Lesson 2 — Vectors: The Language of Movement

What Is a Vector?
A vector is a value with both magnitude (size) and direction. In 3D we write it as three numbers: (x, y, z). Vectors can represent:
  • Position — "the planet is at (200, −50, 400)"
  • Velocity — "the ship moves 80 units/sec forward, 20 units/sec right"
  • Direction — "the ship faces this way" (a unit vector, length = 1)
  • Force/Thrust — "apply 100 N of thrust in this direction"

The Dot Product — "How Aligned Are Two Vectors?"
dot(a, b) = a.x·b.x + a.y·b.y + a.z·b.z
When both vectors are unit vectors (length = 1), the result equals cos(angle between them):
  dot = +1 → same direction     dot =  0 → perpendicular     dot = −1 → opposite
